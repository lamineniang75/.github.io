Introduction

Configuration d’une solution de téléphonie sur IP avec Asterisk (IPBX open-source) et Cisco CallManager Express (CME) pour gérer les téléphones IP.

Configuration

Installation d’Asterisk sur Ubuntu avec codecs Opus/MP3 et langue française. Configuration des comptes SIP et du plan de numérotation via sip.conf et extensions.conf. Cisco CME configuré pour les protocoles SCCP/SIP et le service TFTP.
